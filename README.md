# A1 Grocery & Halal Meat

Static e-commerce pickup website for A1 Grocery & Halal Meat in Binghamton, New York.

## Publish With GitHub Pages

This site is built with plain HTML, CSS, and JavaScript.

Required files:

- `index.html`
- `style.css`
- `script.js`
- `images/`

To publish:

1. Create a GitHub repository.
2. Upload all files from this folder.
3. In GitHub, open **Settings > Pages**.
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ root**
5. Save.

GitHub will give you a public website link after it finishes deploying.

## Product Images

Product photos live in `images/products/`.

Use the exact filenames listed in `images/products/IMAGE-LIST.md`.
